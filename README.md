# Churn_Analysis.py
PROJECT TITLE  
Retail Banking Customer Churn Analysis.

PROJECT OVERVIEW  
This project analyzes customer behavior data from a retail banking dataset to identify key drivers of customer churn. Using Python,pandas,matplotlib, the project focuses on understanding how customer activity, product usage, balance segments, demographics, and geography impact churn. The goal is to generate actionable business insights that help improve customer retention and engagement strategies.

BUSINESS OBJECTIVE  
- Identify high-risk customer segments contributing to churn  
- Analyze customers churn by age, geography, gender, balance segment, activity status    
- Support data-driven decision-making for retention campaigns  
    
DATASET DESCRIPTION  
The dataset contains customer demographic, financial, and behavioral attributes including:  
- RowNumber	
- CustomerId
- Surname	
- CreditScore	
- Geography	
- Gender	
- Age	
- Tenure	
- Balance	
- NumOfProducts
- HasCrCard
- IsActiveMember	
- EstimatedSalary	
- Exited  

TOOLS & TECHNOLOGIES  
- Python (Pandas) 
- Visualization (Matplotlib, Plotly)  
- Data Analysis (EDA)    
- Version Control (Git and GitHub)  
- Notebook (Jupyter Notebook)  

KEY ANALYSIS PERFORMED  
- Customers churn by age groups  
- customers churn by Geography
- Customers churn by activity status
- customers churn by Numbers of product
- customers churend by balance status
